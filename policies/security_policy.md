# Information Security Policy

## Overview

Acme Corporation is committed to protecting company data, customer information, and employee privacy. All employees must follow these security guidelines to prevent data breaches and maintain trust.

## Scope

This policy applies to:
- All employees, contractors, and vendors
- All company systems, devices, and data
- Both on-site and remote work environments

## Password Requirements

### Password Standards
All passwords must meet these requirements:
- **Minimum length**: 12 characters
- **Complexity**: Must include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*)
- **No common patterns**: Avoid dictionary words, your name, or "password"
- **Unique passwords**: Do not reuse passwords across systems

### Password Management
- Change passwords every **90 days**
- Never share passwords with anyone, including IT staff
- Use a company-approved password manager (LastPass Enterprise)
- Enable **Multi-Factor Authentication (MFA)** on all accounts

### Account Lockout
- Accounts lock after **5 failed login attempts**
- Contact IT to unlock accounts
- Automated unlock after 30 minutes
- Repeated lockouts may trigger security review

## Device Security

### Company Devices
- Keep operating system and software updated
- Install only approved software
- Enable full-disk encryption
- Use screen lock (auto-lock after 5 minutes of inactivity)
- Report lost or stolen devices immediately

### Personal Devices (BYOD)
If using personal devices for work:
- Must be enrolled in Mobile Device Management (MDM)
- Must have passcode/biometric lock enabled
- Company reserves right to remote wipe if lost
- Separate work data using company container apps

### USB and External Media
- Do not insert unknown USB devices
- Scan all external media with antivirus before use
- Encrypted USB drives required for sensitive data
- Report suspicious devices to IT Security

## Data Classification

### Confidential Data
Highest level of protection required:
- Customer personal information (PII)
- Financial records
- Trade secrets
- Employee records
- Strategic plans

### Internal Data
For company use only:
- Internal communications
- Project documentation
- Non-public policies
- Business metrics

### Public Data
May be shared externally:
- Marketing materials
- Published product information
- Public announcements

## Data Handling

### Storing Data
- Store confidential data only in approved systems
- Do not save sensitive data to personal devices
- Use encrypted storage for sensitive files
- No confidential data on cloud storage unless approved (approved: Google Drive Enterprise, SharePoint)

### Transmitting Data
- Use encrypted email for sensitive information
- Do not send passwords via email
- Use secure file transfer for large files
- Verify recipient before sending confidential data

### Disposing of Data
- Shred physical documents containing sensitive information
- Use secure delete for electronic files
- Return all company data upon termination
- IT handles device decommissioning

## Network Security

### Company Network
- Connect only company-approved devices
- Do not share network credentials
- Report suspicious network activity
- Guest network for personal devices

### VPN Usage
When working remotely:
- **Always use VPN** to access company resources
- Do not disable VPN for convenience
- Report VPN connection issues to IT
- VPN: vpn.acmecorp.com

### Public WiFi
- Avoid public WiFi for work tasks
- If necessary, use VPN before accessing any company resources
- Never access confidential systems on public WiFi
- Disable auto-connect to open networks

## Email Security

### Phishing Prevention
- Verify sender before clicking links
- Report suspicious emails to security@acmecorp.com
- Do not download unexpected attachments
- When in doubt, contact IT Security

### Email Best Practices
- Do not send sensitive data via email when possible
- Use encryption for confidential communications
- Double-check recipients before sending
- Use BCC for large distribution lists with external parties

## Physical Security

### Office Access
- Always badge in and out
- Do not tailgate or hold doors for unknown persons
- Report lost badges immediately
- Escort all visitors

### Clean Desk Policy
- Lock computers when leaving desk
- Store sensitive documents in locked cabinets
- Shred rather than recycle sensitive papers
- Clear whiteboards after sensitive discussions

## Incident Reporting

### What to Report
- Suspected phishing attempts
- Lost or stolen devices
- Unauthorized access attempts
- Suspicious behavior or persons
- Data breaches or leaks
- Malware or virus infections

### How to Report
- **Email**: security@acmecorp.com
- **Phone**: Security hotline: 1-800-SEC-HELP
- **Portal**: security.acmecorp.com
- **In person**: Approach any IT Security team member

### Response Timeline
- Report incidents within **1 hour** of discovery
- IT Security will respond within **4 hours**
- Do not attempt to investigate on your own
- Preserve evidence (don't delete emails, don't turn off devices)

## Training Requirements

All employees must complete:
- **Security Awareness Training**: Annually
- **Phishing Simulation**: Monthly tests
- **Role-specific training**: As assigned
- **New hire security orientation**: Within first week

## Compliance and Violations

### Compliance
- This policy complies with SOC 2, GDPR, and industry regulations
- Regular audits ensure adherence
- Third-party security assessments conducted annually

### Violations
Policy violations may result in:
- Verbal or written warning
- Mandatory additional training
- Suspension of system access
- Termination of employment
- Legal action (for serious violations)
